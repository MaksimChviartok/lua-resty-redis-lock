Module resty.redis.lock

Simple locking module using redis and ngx_lua

Functions

new (redis, key, ttl)
create a new redis lock.

Parameters:

  • redis resty.redis a resty.redis object
  • key string key to use for locking. The actual redis key will be this string prepended with “LOCK:”
  • ttl number expiry time for the lock. default is 60 seconds

Returns:

    resty.redis.lock a lock object
lock (self, retries, sleep)
try to obtain the lock. Retry until lock is obtained or after a certain number of retries

Parameters:

  • self resty.redis.lock
  • retries number how many time to attempt to obtain the lock. default: 100
  • sleep number how long to sleep between retries. default: 0.010 (10ms). With the defaults, the lock will be retried for 10 seconds

Returns:

  1. boolean lock results. @see try_lock
  2. string error, if applicable
touch (self, ttl)
“touch” the lock. If the lock is held, extend the expire time

Parameters:

  • self resty.redis.lock
  • ttl number how long to extend the lock for. default: value of ttl passed to new

Returns:

  1. boolean success
  2. string error, if applicable
